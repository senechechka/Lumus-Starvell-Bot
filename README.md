## 🤖 **Возможности**

### 🛒 **Starvell**

- Автоответ на заготовленные команды.
- Автовыдача товаров через плагины (например AutoSMM).
- Автоподнятие категорий.
- Автоматическое подтверждение выполнения заказа.
- Уведомления в Telegram.
- Полноценная ПУ в Telegram.

### 🗨️ **Уведомления и ПУ в Telegram**

- Уведомления о новых заказах и оплате.
- Уведомления о подтверждении заказа.
- Уведомления о новых отзывах.
- Уведомления о новых сообщениях.
- Возможность полностью настраивать автоответчик и остальные модули LSB.

### ⚙️ **Дополнительные возможности**

- Использование переменных в тексте для автоответа.
- Создание плагинов для кастомизации функционала без редактирования исходного кода самого бота.

## 🥇 **Преимущества**

### 😀 **Для пользователей**

- **Больше**, чем наличие самого нужного функционала.
- Гибкие и при этом простые конфиги, написанные в INI/JSON-формате.
- Полное управление через Telegram.

### 💻 **Для разработчиков**

- Python с современным синтаксисом (type-хинты, `from __future__ import annotations`).
- Широкое использование ООП. Каждое событие (новое сообщение, заказ, оплата, подтверждение, отзыв) — отдельный класс, а не просто словарь.
- Возможность легкого создания плагинов через систему биндингов (`BIND_TO_NEW_ORDER`, `BIND_TO_PAYMENT` и т.д.).
- Сконфигурированный логгер с цветным выводом.
- Собственный Python-пакет StarvellAPI.

## 🔌 Плагины

- AutoSMM — автоматическая накрутка через SMM-сервисы (twiboost и аналоги).
- Advanced Profile Stat — расширенная статистика профиля.

## ⬇️ Установка

### 🔷 Windows

1. Скачайте и установите Python 3.11+.
   1. При установке поставьте галочку у `Add python.exe to PATH` на первом экране установки.
2. Скачайте Lumus Starvell Bot.
3. Перенесите архив в нужное вам место и распакуйте его.
4. Перейдите в папку проекта.
5. Запустите файл `Setup.bat`. Дождитесь окончания загрузки пакетов.
6. Закройте командную строку и запустите файл `Start.bat`.

## 🔌 Установка плагинов

Не устанавливайте плагины из непроверенных источников. Через систему плагинов злоумышленники могут получить полный доступ к вашему устройству или аккаунту Starvell.

1. Поместите файл плагина (`.py`) в папку `plugins/`.
2. Перезапустите бота — плагин загрузится автоматически.
